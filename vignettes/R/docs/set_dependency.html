set_dependency {WorkflowRender} R Documentation

function for create application dependency data

Description

Usage

set_dependency(
    context.env = 'list'(),
    workfiles = 'list'());

Arguments

context.env

a list of the depends environment symbol in the current workflow context, value of this parameter it could be: 1. a character vector of the required symbol(will assume that no data type is required), or 2. a tuple list object with key name is the symbol name and the corresponding tuple value is the required symbol type.

workfiles

a set of the required temp result files in the workflow the format of this parameter value is a tuple list of the file path, where the key name is the app name and the corresponding tuple list value is a character vector of the required multiple reference temp file. the data format of this parameter would be: list(app1 = [file1, file2, ...], app2 = file3, ...)

Details

the allowed data type of the context symbol could be: any, numeric, integer, character, logical, function

Authors

xieguigang

Value

This function has no value returns.
