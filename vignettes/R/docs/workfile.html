workfile {WorkflowRender} R Documentation

Construct of the file path inside an app workspace

Description

Usage

workfile(app,
    relpath = NULL);

Arguments

app

this app parameter value could be in 3 kinds of data type: 1. app tuple list object, which is created via the app function. 2. the app name character vector 3. the workfile path expression, should be in format like: app://relpath/to/file.ext

Details

the workfile path expression is in format string of: app://filepath, example as getMzSet://mzset.txt, where we could parse the reference information from this character string: app name is getMzSet, and the relpath data is mzset.txt. so, such configuation could be equals to the function invoke of the workfile function: workfile("getMzSet", "/mzset.txt");.

Value
