workspace_temproot {WorkflowRender} R Documentation

Get the Workspace Temporary Root Directory

Description

Returns the temporary root directory of the current workflow workspace. This directory holds all of the intermediate files produced by the workflow modules, including the per-application workspace directories returned by \code{\link{workspace}}.

Usage

workspace_temproot(
    ssid = NULL);

Arguments

ssid

the session identifier used for the multi-user environment. The default value of \code{NULL} refers to the global user environment.

Details

The returned path corresponds to the \code{temp_dir} slot of the workflow context, which is initialized by \code{\link{init_context}} to \code{/tmp/}.

Authors

xieguigang

Value

A character vector of the temporary root directory path of the workflow workspace.

Examples

#\dontrun{

tmp <- workspace_temproot();

#}
