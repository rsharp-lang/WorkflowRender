filepath_safe {WorkflowRender} R Documentation

Create the file path safely

Description

Usage

filepath_safe(dir, filename,
    maxchars = 48);

Arguments

Details

this function will make a safe file path, by processing the file name string in process:

  1. removes the unsafe character for windows filesystem
  2. make the file name short if the file name string len exccede the max chars

Authors

xieguigang

Value

this function generates a new character vector based on the input dir and the filename for save a file safely.
