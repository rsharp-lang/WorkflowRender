use_cache {WorkflowRender} R Documentation

Work with cache file

Description

Usage

use_cache(filepath, create);

Arguments

create

a callable function that accept a file path parameter to create the target cache file.

Details

this function will work around with the specific given cache file: this function will do nothing if the cache file is already exists or invoke the create function for when file exists of the cache data is false.

Authors

xieguigang

Value

This function has no value returns.
