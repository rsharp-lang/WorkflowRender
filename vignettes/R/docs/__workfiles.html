__workfiles {WorkflowRender} R Documentation

Internal Function for Parsing the Workfile URI Expression

Description

Wraps the actual URI parser \code{\link{workfiles_data}} with a short-circuit for the empty-input case. This function is invoked internally by \code{\link{build_app}} to convert the \code{@workfiles} attribute of a tagged function into the structured dependency list expected by \code{\link{set_dependency}}.

Usage

__workfiles(files.uri);

Arguments

files.uri

a character vector of workfile URI expressions in the format \code{app://relpath/to/file.ext}. May be \code{NULL} or empty.

Details

Authors

xieguigang

Value

A named list object that groups the parsed file paths by their producing application name, suitable for direct use as the \code{workfiles} argument of \code{\link{set_dependency}}. When \code{files_uri} is empty, \code{NULL} is returned instead.
